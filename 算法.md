## 欧几里得算法，辗转相除法
* 辗转相除法，是指用于计算两个正整数a，b的最大公约数。
* 定理：两个整数的最大公约数等于其中较小的那个数和两数相除余数的最大公约数。最大公约数（Greatest Common Divisor）缩写为GCD。
* gcd(a,b) = gcd(b,a mod b) (不妨设a>b 且r=a mod b ,r不为0)
```cpp
int divisor(int m,int n)
{
    if (m % n == 0) {
        return n;
    }
    else {
        return divisor(n,m % n);
    }
}
```
